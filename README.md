# End-to-End-Customer-Churn-Analysis-using-Python-and-Power-BI
End-to-end Customer Churn Analysis using Python, Pandas, and Power BI with interactive dashboards and business insights.

## Project Overview
This project analyzes customer churn using Python (Pandas) and Power BI. It includes data cleaning, exploratory data analysis (EDA), and an interactive dashboard that helps visualize customer behavior, subscription trends, revenue, and churn metrics.

##  Tools & Technologies

- Python
- Pandas
- Jupyter Notebook
- Power BI
- DAX

## Dashboard Features

- Total Customers
- Monthly Revenue
- Average Monthly Charges
- Average CLTV
- Average Churn Score
- Plan Type Analysis
- Contract Type Analysis
- Renewal Trend
- Interactive Filters (Gender, Plan Type, Contract Type)
